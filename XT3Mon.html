BigBen Cray XT3 Monitor Applet

This applet grabs the latest XT3 data files over HTTP, and therefore permissions must be set to allow it to do so. Add the following to your ~/.java.policy file: 
	grant codeBase "http://www.psc.edu/~yanovich/XT3Mon" {
		permission java.net.SocketPermission "bigben-monitor.psc.edu", "resolve";
		permission java.net.SocketPermission "bigben-monitor.psc.edu:80", "connect";
		permission java.net.SocketPermission "bigben-monitor.psc.edu:80", "resolve";
	};
